# Butty Core

First spike to show the comment function on files. 
This is very rudimentary. The method of how the comments are saved, will change.

## Installation

`npm install`

## Start

`npm start`

## Development

Run the Tests:

`npm test`

Rebuild the documentation:

`npm run-script docs`
